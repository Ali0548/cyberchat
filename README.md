# ChatGPT Web App Clone

## Installation

Clone repo:

```bash
git clone https://github.com/dan-kwiat/chat-gpt-clone.git
cd chat-gpt-clone
```

Install deps:

```bash
yarn
# Or
npm install
```

Set local env:

```bash
cp .env .env.local
# Now edit `.env.local` with your API key from OpenAI
```

## Development

```bash
yarn run dev
```
